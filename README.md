# API Barbería - Proyecto Formativo

Este proyecto contiene el desarrollo de servicios web (API REST) para la gestión de una barbería.

## Estructura
- `/api`: Contiene el script `api.php` para la conexión y CRUD.
- `/database`: Contiene el archivo SQL con la estructura de la base de datos.

## Tecnologías
- PHP
- MySQL
- html5
- Css3
- javascrip


##  Pruebas de Integración
He incluido la colección de Postman `Texting.postman_collection.json`. 
Para validar los servicios:
1. Importa este archivo en Postman.
2. Ejecuta los métodos GET, POST, PUT y DELETE.
3. Verifica los cambios en MySQL Workbench.

# 💈 BarberiaManagementSystem - Sistema de Gestión de Citas

Este proyecto es una solución integral para la administración de una barbería, desarrollado como un sistema de gestión web que permite el control de citas, clientes y servicios. El proyecto sigue una arquitectura de API RESTful para el manejo de datos y una interfaz responsiva diseñada con los estándares modernos de CSS y frameworks de diseño.

---

## 🛠️ Stack Tecnológico

El desarrollo de este sistema se basa en las siguientes herramientas y tecnologías:

* **Entorno de Servidor:** Laragon v7.0.6 (Localhost)
* **Gestión de Base de Datos:** MySQL (Motor de base de datos)
* **Visualización de BD:** MySQL Workbench
* **Lenguaje Backend:** PHP
* **Control de Versiones:** Git
* **Frontend:** HTML5, CSS3 (Media Queries), Bootstrap v5, Font Awesome
* **Diseño UX/UI:** Figma
* **Pruebas de API:** Postman
* **Entorno de Desarrollo:** VS Code / Antigravity
